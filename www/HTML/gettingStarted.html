ProVision: Getting started

Requirements

ProVision is designed for statistical analysis of label free and tandem mass tagged proteomics data that was generated using the popular maxQuant computational proteomics platform. The only requirement to use ProVision is a file called protienGroups.txt. If your samples were not searched by maxQuant and you would like to use this platform, you can find how to do the searches yourself here.Finally to export to Excel you will need updated java on your system. To get java, download it here

Default parameters

Pre processing of output data is important to the downstream statistical tests and accurate visulisation but may also be the most unintuitive step of the pipeline. Default parameters are pre chosen to yeild optimal results and can simply be followed as is. Options are however in place to alter the pre chosen values if that is desired. The full tutorial will walk through the use of ProVision step by step and explain as much as possible as we go.

Reactivity

We have designed the application with a high degree of reactivity. Choices as far as the first fitlering steps will propagate throughout the program and influence the final graphs. Thus it is possible to go through the data analysis process and come back to the data processing page and alter the parameters to obtain a different result. This may be usefull if the initial parameters are too lenient and a large amount of proteins are statistically significant.
